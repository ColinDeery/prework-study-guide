HTML

  • HTML provides content to a webpage.
  • The head element contains information and metadata about the webpage.
  • The body element represents the visible content shown to the user.
  • Section elements are semantic elements that define a section in a document that contains thematic content.
  • A section element will allow us to organize content on a page based on topics we have.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Declarations are comprised of a property and the value assigned to it.
  • A selector defines the element or attributes to which declarations are applied

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch name: creates a new branch and switches to it

JavaScript

  • A JavaScript variable is a named container that allows us to store data in our code.
  • The console.log method is used to to output a message to the web console by adding an argument.
  • Console refers to a test environment that developers use to check out their code, while .log() is a method which contains a set of instructions that can be exectued by a computer.
